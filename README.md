# This repo contains a simple page counter with Redis and Flask

### Requirements 
- Install [Vagrant](https://www.vagrantup.com/)
- Start project
    ```
    git clone https://github.com/chavo1/python-page-counter.git
    cd python-page-counter
    vagrant up
    ```
- view the app at:
    ```
    http://192.168.56.56:5000
    ```
